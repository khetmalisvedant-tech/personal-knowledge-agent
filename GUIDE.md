# 🧠 Personal Knowledge Base Agent — VS Code Setup Guide

> **Stack:** Python · FastAPI · Google Gemini 2.5 Flash · Vanilla JS UI

---

## 📁 Project Structure

```
pkb_agent/
├── .vscode/
│   ├── launch.json       ← F5 debug config (runs uvicorn automatically)
│   ├── settings.json     ← Python interpreter + formatting
│   └── extensions.json   ← Recommended VS Code extensions
│
├── agent.py              ← Core AI agent (search, RAG, clustering)
├── main.py               ← FastAPI server (API routes + UI serving)
├── index.html            ← Web UI (dark theme, all pages)
│
├── .env.example          ← Template — copy to .env
├── .env                  ← Your secrets (NOT committed to Git)
├── .gitignore
├── requirements.txt
└── pkb_data.json         ← Auto-created when you save your first note
```

## ✅ Step-by-Step Setup

### Step 1 — Open in VS Code

```
File → Open Folder → select the pkb_agent folder
```

Or from terminal:
```bash
code pkb_agent
```

---

### Step 2 — Install Recommended Extensions

When VS Code opens, you'll see a popup:
> *"This workspace has extension recommendations"*

Click **Install All**. This installs:
- `ms-python.python` — Python language support
- `ms-python.debugpy` — Debugger (needed for F5)
- `ms-python.black-formatter` — Auto-formatting
- `rangav.vscode-thunder-client` — Test API endpoints inside VS Code

---

### Step 3 — Create a Virtual Environment

Open the **VS Code Terminal** (`Ctrl+`` ` `` ` or `View → Terminal`) and run:

**Windows:**
```bash
python -m venv venv
venv\Scripts\activate
```

**Mac / Linux:**
```bash
python3 -m venv venv
source venv/bin/activate
```

You should see `(venv)` appear at the start of your terminal prompt.

---

### Step 4 — Install Dependencies

With the venv activated:
```bash
pip install -r requirements.txt
```

This installs:
| Package | Purpose |
|---|---|
| `google-generativeai` | Gemini 2.5 Flash API client |
| `fastapi` | Web framework |
| `uvicorn` | ASGI server (runs FastAPI) |
| `python-multipart` | File upload support |
| `python-dotenv` | Loads `.env` file automatically |

---

### Step 5 — Get Your Gemini API Key

1. Go to → **https://aistudio.google.com/app/apikey**
2. Sign in with your Google account
3. Click **"Create API Key"**
4. Copy the key (starts with `AIza...`)

> **Free tier:** 15 requests/min · 1,000,000 tokens/day — plenty for personal use.

---

### Step 6 — Set Up Your `.env` File

In the project root, copy the example file:

**Windows:**
```bash
copy .env.example .env
```

**Mac / Linux:**
```bash
cp .env.example .env
```

Open `.env` and paste your key:
```env
GEMINI_API_KEY=AIzaSy...your_actual_key_here...
```

> ⚠️ Never share or commit `.env`. It's already in `.gitignore`.

---

### Step 7 — Select the Python Interpreter

Press `Ctrl+Shift+P` → type **"Python: Select Interpreter"** → choose the one that says:

```
Python 3.x.x ('venv': venv)
```

This tells VS Code to use the virtual environment you created.

---

### Step 8 — Run the App

**Option A — Press F5** (recommended)
- VS Code reads `.vscode/launch.json` and starts the server with the debugger attached
- You can set breakpoints in `agent.py` or `main.py` and they will pause execution

**Option B — Terminal**
```bash
python main.py
```

**Option C — uvicorn directly**
```bash
uvicorn main:app --reload --port 8000
```

---

### Step 9 — Open the App

Visit: **http://localhost:8000**

The web UI loads with 7 pages:
| Page | What it does |
|---|---|
| Dashboard | Stats overview — note count, word count, top tags, recent notes |
| Add Note | Save a new note with title, content, and tags |
| All Notes | Browse, preview, and delete every note |
| Search | Semantic TF-IDF search across your knowledge base |
| Ask AI | Type a question → Gemini answers using only your notes (RAG) |
| Topics | AI clusters your notes into themes with labels + summaries |
| Upload File | Drag & drop `.txt` or `.md` files to import |

## ⚙️ How It Works (Architecture)

```
Browser (index.html)
        │  HTTP fetch()
        ▼
FastAPI Server (main.py)   ← .env loaded here
        │  calls
        ▼
PKBAgent (agent.py)
   ├── CRUD → pkb_data.json
   ├── TF-IDF Search (pure Python, no ML libs)
   ├── KMeans Clustering (pure Python)
   └── Gemini 2.5 Flash API
              │
              ▼
        google.generativeai
```

### Search (TF-IDF)
All notes are tokenized and converted to TF-IDF vectors. Search queries are vectorized the same way. Notes are ranked by **cosine similarity** — no external vector database needed.

### RAG Q&A
1. Search notes for the top-5 most relevant chunks
2. Pack them as context into a Gemini prompt
3. Gemini answers **only from your notes** and cites sources

### Topic Clustering
1. Vectorize all notes (TF-IDF)
2. KMeans groups them into clusters
3. Gemini generates a label and 1-2 sentence summary per cluster

## 🐛 Debugging in VS Code

With F5, you have full debugger access:

1. **Set a breakpoint** — click the red dot left of any line number in `agent.py`
2. **Press F5** — the server starts; when that line is hit, execution pauses
3. **Inspect variables** — hover over any variable or use the **Variables** panel
4. **Step through** — use F10 (step over) / F11 (step into) / F5 (continue)

Useful lines to put breakpoints:
- `agent.py` line with `_ask_gemini(prompt, ...)` — inspect the prompt sent to Gemini
- `agent.py` line with `_kmeans(matrix, k)` — inspect clustering inputs
- `main.py` in `ask()` — inspect the incoming question

## ❓ Troubleshooting

| Problem | Fix |
|---|---|
| `EnvironmentError: GEMINI_API_KEY is not set` | Make sure `.env` exists and has your key |
| `ModuleNotFoundError: No module named 'fastapi'` | Run `pip install -r requirements.txt` with venv activated |
| Port 8000 already in use | Change `PORT=8001` in `.env` or kill the other process |
| `401 API_KEY_INVALID` from Gemini | Double-check your key at https://aistudio.google.com/app/apikey |
| F5 doesn't start | Ensure `ms-python.debugpy` extension is installed and venv interpreter is selected |
| Notes not saving | Check write permissions in the project folder; `pkb_data.json` is created automatically |
| Clustering returns 1 cluster | Add at least 4-5 notes on different topics first |
